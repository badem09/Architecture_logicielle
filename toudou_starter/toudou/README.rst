Toudou
======

The best todo application!


Contributing
------------

Start coding using an editable project with:

.. code-block:: text

    $ python3 -m venv venv
    $ source ./venv/bin/activate
    (venv) $ pip install -e .
    (venv) $ toudou

Or with PDM:

.. code-block:: text

    $ pdm install
    $ pdm run toudou
